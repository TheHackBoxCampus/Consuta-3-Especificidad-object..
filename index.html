Mobile First

" Dar prioridad al desarrollo del producto para smartphones y otros dispositivos móviles frente al de los tradicionales PC "

" De esta forma las características diferenciadoras de una app pueden convertirse en una de las claves del éxito de una startup. Por ello para algunos negocios no es necesario desarrollar una web o una aplicación compleja para el ordenador desde el primer momento, sino que crear una potente app móvil desde el principio siguiendo la estrategia mobile first puede marcar la diferencia. "
